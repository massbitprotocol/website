Why Massbit?

Massbit mission is to become an innovative infrastructure to make DeFi and Web3 development easier, faster and cheaper. We first look to build decentralized APIs and cross-chain indexing.

Decentralized APIs for blockchains

We work with small API providers and validators to form decentralized gateways. Some API providers we are currently working with only only utilize 20% of their capacities while validators or miners only do their work to earn rewards but waste other resources (data and apis). That gives them incentives to work with MassBit to earn more rewards and serve DeFi and Web3 apps avoid better and cheaper. as relying one single entity causes centralization problem and privacy concern.

Cross-chain indexing

Blockchain indexing is essential to Defi and Web3 development. It does not make sense for every single Web3 developer to build the whole pipeline of running a node, getting, filtering the right data, indexing and serving the data. It also takes lots of effort to build the pipelien stable and reliable. MassBit provides the infrastructure to index data of different blockchains like Substrate, Ethereum and especially fast blockchains like BSC, Matic and Solana. Developers only do 10% of the work as write the indexing logic in different language (Rust, AssemblyScript, TypeScript).

Massbit Platform

Massbit incentivizes the community to run the infrastructure in the decentralized fashion. The core verification protocol is to make sure the participants act honestly. By the protocol, in order to join MassBit working, they have to stake MBT tokens, do the work properly and verify each others to get rewards. Otherwise, they will get slashed, get penalized or lose their staked MBT tokens
